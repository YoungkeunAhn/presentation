슬라이드에서 제거한 긴 프롬프트들은 삭제하지 말고 별도 섹션으로 분리해줘.

가능한 방식:

- `appendix` 섹션으로 HTML 하단에 추가
- 또는 별도 `prompts.html` 파일로 분리
- 또는 화면에서는 숨겨두고 인쇄/복사용으로 볼 수 있게 구성

실습 프롬프트 모음에 포함할 것:

1. 기획 문서 요청 프롬프트
2. Supabase SQL 요청 프롬프트
3. 구현 요청 프롬프트
4. 검증 체크리스트 요청 프롬프트
5. 에러 해결 요청 프롬프트
6. 내 아이디어로 바꾸기 프롬프트

중요:

- 발표 슬라이드에는 요약 프롬프트만 보여줘.
- 수강생이 복사해서 쓸 긴 프롬프트는 실습 프롬프트 모음에서 제공해줘.
- 이렇게 해서 슬라이드는 간결해지고, 실습 자료는 실용적으로 남게 해줘.
