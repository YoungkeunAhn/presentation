현재 오프닝 쪽에는 표지, 오늘의 약속, 학습 목표, 전체 흐름 등이 따로 나뉘어 있는데, 일부 내용이 겹쳐 보여.  
오프닝은 3장 정도로 압축해줘.

## 오프닝 1장: 표지

역할:

- 강의의 주제를 한 번에 보여주는 첫 화면

추천 제목:  
`AI에게 일을 맡기고 결과를 확인하는 첫 경험`

추천 부제:  
`완벽한 이해보다 중요한 것은 작게 요청하고, 화면을 보고, 다시 고쳐달라고 말하는 감각입니다.`

유지할 요소:

- 강의명
- 초급자 실습
- AI와 작은 결과물 만들기

문구는 너무 길지 않게 유지해줘.

## 오프닝 2장: 오늘의 약속

현재 `오늘의 약속` 슬라이드는 유지하되 문구를 더 짧게 다듬어줘.

추천 제목:  
`전문가가 되는 시간이 아니라, 덜 막막해지는 시간`

카드 3개:

1. 질문이 서툴러도 괜찮다
    - 처음 쓰는 도구 앞에서 막막한 것은 자연스럽습니다.
2. 결과물이 단순해도 괜찮다
    - 오늘은 완성도보다 한 번 끝까지 해보는 경험이 중요합니다.
3. 다시 요청하면 된다
    - AI의 첫 답변은 초안이고, 우리는 결과를 보고 고칩니다.

## 오프닝 3장: 오늘의 흐름

현재 `학습 목표`와 `전체 흐름`은 하나로 합쳐줘.  
둘 다 강의 전체 방향을 안내하는 역할이므로 따로 둘 필요가 없어.

추천 제목:  
`오늘 익힐 AI 활용 흐름`

추천 부제:  
`막막함을 줄이고, 요청하고, 작게 만들고, 다시 개선합니다.`

4단계 흐름:

1. 덜 막막해지기
    - AI에게 무엇을 맡길 수 있는지 감을 잡습니다.
2. 잘 요청하기
    - 상황, 목표, 결과물, 조건을 담아 요청합니다.
3. 작게 만들기
    - 기획 → 설계와 문서화 → 구현 → 검증을 한 번 경험합니다.
4. 다시 개선하기
    - 부족한 결과와 에러를 AI에게 다시 전달합니다.

하단 문구:  
`요청 → 확인 → 다시 요청`

기존 `전체 흐름` 슬라이드의 오프닝/1부/2부/3부/마무리 타임라인은 삭제하거나 이 슬라이드에 흡수해줘.
