최종 결과물은 아래 형태로 만들어줘.

1. 압축된 발표용 HTML 슬라이드
    - 22~26장, 가능하면 24장 내외
    - 기존 43장보다 훨씬 간결한 구조
    - 강의 흐름이 끊기지 않게 구성
2. 실습 프롬프트 모음
    - 긴 프롬프트를 별도 섹션 또는 별도 파일로 정리
    - 수강생이 복사해서 쓸 수 있게 구성
3. 수정 요약
    - 몇 장에서 몇 장으로 줄였는지
    - 어떤 슬라이드를 합쳤는지
    - 어떤 내용을 실습 프롬프트 모음으로 분리했는지
    - 파트별 최종 슬라이드 구성을 간단히 요약

최종 HTML은 바로 브라우저에서 열어 발표할 수 있어야 하고, 기존 슬라이드 이동 기능은 유지해줘.
